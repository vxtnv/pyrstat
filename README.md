
# pyrstat

R econometrics in Python – powered by rpy2 + rpackagemanager.

## Requirements

- Python 3.11+
- R installed locally
- rpy2 (ABI mode)

## Installation

pip install --force-reinstall "git+https://github.com/vxtnv/pyrstat.git"



## R Packages Used

sandwich, lmtest, car, ivreg, forecast, tseries, aTSA, vars, urca, zoo

All R packages are installed automatically via [rpackagemanager](https://github.com/vxtnv/RPackageManager).
